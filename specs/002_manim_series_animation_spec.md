# Manim 仕様書: 等差数列の和アニメーション

## 概要

**ファイル名:** `series_animation.py`  
**クラス名:** `SeriesAnimation`  
**解像度:** 1080×1920（9:16）、フレームレート: 60fps  
**所要時間（目安）:** 約80〜100秒（Round 1 + Round 2）  
**テーマ:** 背景 `#0d1117`（ほぼ黒）、テキスト `#e6edf3`（オフホワイト）

## 全体構成

アニメーションは **2ラウンド** を順番に実行する。

| ラウンド | 対象 | 証明する等式 |
|---------|------|------------|
| Round 1 | 数値例 | `1+2+3+⋯+10 = 55` |
| Round 2 | 一般式 | `1+2+3+⋯+n = n(n+1)/2` |

---

## レイアウト定義

```
画面中央を原点(0, 0)とするManimデフォルト座標系

Row Y 座標（画面上から）:
  row1_y = 1.2   ← 上段（正順の式）
  row2_y = 0.0   ← 中段（逆順の式）
  row3_y = -1.2  ← 下段（2S(n) = ...）

列構成（数列グリッド）:
  9列: [数, +, 数, +, 数, +, ⋯, +, 数]
  各列幅は均等（合計幅 ≈ 8.0 units）
  グリッド全体は水平方向に中央揃え
```

## Round 1: S(10)の数値例

### Scene 1: 数列の登場と「= ?」

**目的:** `S(10) = 1+2+3+⋯+10 = ?` を提示する

1. row1の位置に `1 + 2 + 3 + ⋯ + 10` を1文字ずつタイプアウト（各要素0.11秒間隔）
2. 末尾に `= ?` をフェードイン
3. 0.9秒静止
4. `= ?` をフェードアウトしながら、左側の `S(10) =` プレフィックスをフェードイン（同時、0.8秒）
5. 0.3秒静止

> **Manim実装メモ:** `Write` または `FadeIn` + `lag_ratio` を使用。プレフィックス `S(10)` は `Text`、`=` と数列要素は `MathTex` または `Text`。

---

### Scene 2: row2の登場（スライドイン）

**目的:** 逆順の式 `S(10) = 1+2+3+⋯+10` を row1 の位置から中段へスライド

1. row1 と同じ内容（`S(10) = 1+2+3+⋯+10`）を row1 と同じ Y 座標に配置・フェードイン
2. `AnimationGroup` で row2_y（中段）へ `MoveToTarget` で 0.9秒スライド

---

### Scene 3: row2 を逆順に反転

**目的:** `S(10) = 1+2+3+⋯+10` → `S(10) = 10+9+8+⋯+1` に変換

1. row2 のグリッド部分を Y 軸で `Rotate(90°)` + フェードアウト（0.42秒）
2. 内容を `10+9+8+⋯+1` に差し替え
3. `Rotate(-90°)` + フェードイン（0.42秒）

> **Manim実装メモ:** `Transform` または `ReplacementTransform` で各要素を対応する逆順要素に変換する方が自然。

---

### Scene 4①: S(10) を囲むボックスと「+」記号

**目的:** 2行の `S(10)` を青いボックスで囲み、「+」で結ぶことを示す

1. row1 と row2 の `S(10)` 部分を包む **角丸矩形**（`RoundedRectangle`）をフェードイン
   - 色: `#58a6ff`、線幅: 2.5、fill: `rgba(88,166,255,0.09)`
2. 2つの `S(10)` の中間に赤い `+` 記号をフェードイン（`#ff7b72`）
3. 0.65秒静止
4. `+` 記号をフェードアウト

---

### Scene 4②③: 列ごとにペアの和を表示

**目的:** 各列（1+10、2+9、3+8、⋯）を順にハイライトして `11` を積み上げる

row3 の位置に `2S(10) = [resArea]` を配置する。

各列（5組: col0, col2, col4, col6=⋯, col8）について以下を繰り返す:

1. row1 と row2 の当該列を青いボックスで囲む（`⋯` 列はボックスなし）
2. resArea に `+` (2組目以降) と値 (`11` or `⋯`) をフェードイン
   - `11` は `#7ee787`（グリーン）
3. ボックスをフェードアウト
4. 次の列へ

---

### Scene 5: グレーアウト → 波括弧 → 右辺変換

**目的:** `11×10 = 110` を導出する

1. row1・row2 を opacity 0.2 にフェードアウト（薄暗く）
2. resArea（`11+11+⋯+11`）の下に **波括弧（アンダーブレース）**と `10個` ラベルを表示
   - 色: `#ff9d5c`（オレンジ）
   - `Brace` オブジェクト（Manimの `BraceLabel`）を使用
   - 0.95秒後にフェードアウト
3. resArea の内容を `11 × 10` にフェード切替（0.8秒）
4. `11 × 10` を `110` にフェード切替（0.7秒）、`#ffd700`（ゴールド）で表示

---

### Scene 6: 中央へ収束 → ÷2 → S(10)=55

**目的:** `2S(10) = 110` から `S(10) = 55` を導出する

1. row1・row2 をフェードアウト（0.4秒）
2. `2S(10) = 110` を画面中央へスムーズにアニメーション移動（0.65秒）
3. `110` の右に `÷2` をフェードイン、`2` に打ち消し線（`#ff7b72`）を描画（同時）
4. 0.75秒後、`2`（打ち消し線ごと）・`÷2`・`110` をフェードアウト
5. `=` の直後に `55` をフェードイン（`#e6edf3`）
6. `55` を **ゴールドにグロー強調**（1.4秒）
   - `color: #ffd700`
   - `GlowEffect` または `Flash` アニメーション（外側に放射）

---

### Scene 7: 最終式の表示

**目的:** `1+2+3+⋯+10 = 55` をクリーンに表示

1. `S(10) = 55` ラッパーをフェードアウト（1.2秒）
2. 画面中央に `1 + 2 + 3 + ⋯ + 10 = 55` をフェードイン（1.4秒）
3. Round 1 は次ラウンドへの接続なので、1.5秒後にフェードアウト

---

## Round 2: S(n) の一般式（同一シーケンスを記号で繰り返す）

Round 1 と全く同じシーン構成で、以下の値に変更する:

| 項目 | Round 1 | Round 2 |
|------|---------|---------|
| ラベル | `S(10)` | `S(n)` |
| 正順 | `1+2+3+⋯+10` | `1+2+3+⋯+n` |
| 逆順 | `10+9+8+⋯+1` | `n+(n-1)+(n-2)+⋯+1` |
| ペア和 | `11` | `(n+1)` |
| 個数ラベル | `10個` | `n個` |
| 掛け算 | `11 × 10` | `(n+1) × n` |
| 2倍の値 | `110` | `n(n+1)` |
| 半分の値 | `55` | `\frac{1}{2}n(n+1)` |
| 最終式答 | `55` | `\frac{1}{2}n(n+1)` |

**Round 2 のみの変更点:**
- ペア和 `(n+1)` はフォントサイズを小さく（約 18px 相当）してグリッド列幅に収める
- `\frac{1}{2}n(n+1)` は `MathTex` で描画
- Round 2 は最後のラウンドなので、最終式 `1+2+⋯+n = \frac{1}{2}n(n+1)` は **フェードアウトせず残す**（ゴールドにグロー強調して保持）

## Manim 実装指針

### 推奨クラス・関数

```python
from manim import *

class SeriesAnimation(Scene):
    def construct(self):
        self.camera.background_color = "#0d1117"
        # Round 1
        self.run_once(config_numeric)
        self.wait(0.6)
        self.clear()
        # Round 2
        self.run_once(config_symbolic, is_last=True)
```


## 注意事項・実装上のポイント

1. **MathTex vs Text の使い分け**  
   - `S(10)`、`S(n)` → `MathTex(r"S(10)")` で描画（斜体・数学フォント）  
   - `+`、`=`、数値 → `MathTex` または `Text`  
   - `\frac{1}{2}n(n+1)` → 必ず `MathTex`

2. **グリッドの位置揃え**  
   - 正順・逆順・結果行の `=` の右端が同じ X 座標になるよう揃える  
   - `VGroup.arrange()` または手動 `next_to` で制御

3. **打ち消し線の描画**  
   - `Line` オブジェクトを `2` の `MathTex` の上に対角線で描画  
   - `Create(line)` でアニメーション

4. **グロー効果**  
   - `Flash(mobject, color="#ffd700", flash_radius=0.5)` で代用  
   - または `mobject.set_color("#ffd700")` + 外側に薄いコピーを置いてフェードアウト

5. **波括弧**  
   - `BraceLabel` または `Brace` + `Text` で実装  
   - 下向き（`DOWN`）、色 `#ff9d5c`

6. **フォント**  
   - Manim のデフォルト LaTeX フォント（Computer Modern）を使用  
   - テキスト系は `font="STIX Two Math"` または `font="Times New Roman"` を指定

7. **アニメーションのクリア**  
   - ラウンド間は `self.remove(*self.mobjects)` または `self.clear()`
