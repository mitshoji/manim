# Manimコード生成指示書：ピタゴラスの定理の証明

## 概要

ピタゴラスの定理 **a² + b² = c²** を、面積の等式として視覚的に証明するアニメーション。
「(a+b)² の正方形から4つの三角形を取り除くと c² が残る」という構造で証明する。

---

## シーン構成（フェーズ順）

### フェーズ1：直角三角形の登場

- 画面中央やや下に、直角三角形を1つフェードインで表示する
- 辺のラベル：直角を挟む2辺を **a**（青）、**b**（緑）、斜辺を **c**（赤）とする
- 直角マーク（□）を直角の頂点に表示する
- 辺ラベルは少し遅れてフェードインする

### フェーズ2：4つの三角形を配置する

- 元の三角形を4枚コピーし、それぞれを回転・移動させて **(a+b) × (a+b) の正方形** の四隅に配置する（すべて鏡像ではなく回転のみ）
  - 元の三角形：時計回り90°回転して左上コーナーへ移動
  - コピー1：180°回転して右上コーナーへ移動
  - コピー2：反時計回り90°回転して右下コーナーへ移動
  - コピー3：回転なし（元と同じ向き）で左下コーナーへ移動
- 各三角形が順番にアニメーションで飛んでいく（1枚ずつ）
- 配置後、4枚の三角形が囲む内側に **斜辺 c を辺とする正方形**（赤枠）が自然に現れる

### フェーズ3：外側の正方形を認識させる

- 外側の大きな正方形（辺 = a+b）を黄色の枠線でハイライトする（フラッシュ演出）
- 上辺と右辺に **a+b** のブレース（波括弧）と寸法ラベルを表示する
  - 上辺：左から **a**（青）、**b**（緑）に分割されていることを示す目盛りを表示
  - 右辺：上から **a**（青）、**b**（緑）に分割されていることを示す目盛りを表示

### フェーズ4：(a+b)² の式が登場

- 外側正方形の中央に **(a+b)²** という式をフェードインで表示する
- その式がスライドして画面下部の数式エリアの行1左端に移動する
- 続いて **= a² + 2ab + b²** が右側に順番に出現する（a²→+→2ab→+→b²の順）
- 色分け：a²は青、b²は緑、2abは白、(a+b)²は黄色、記号は白

### フェーズ5：4つの三角形の面積を示す

- 左上の三角形（tri[0]）だけを白くフラッシュさせて注目させる
- 辺 **a** を青く強調（グロー）→辺 **b** を緑く強調（グロー）の順で光らせる
- 三角形の中央に **½ab** というラベルをフェードインで表示する
- 残り3つの三角形にも同様に **½ab** を一斉に表示する

### フェーズ6：4つの三角形の合計面積

- 4つの三角形の中央付近に **4 × ½ab = 2ab** という式をフェードインで表示する
- この **2ab** がスライドして数式エリアの行2（行1の少し下）の中央付近に移動する
- 移動と同時に元の式は消えていく

### フェーズ7：−2ab の登場と相殺

- 行2の **2ab** の左に **−（マイナス符号）** がフェードインで出現し **−2ab** になる
- 外側正方形の辺ラベル（ブレース）がフェードアウトする
- **−2ab** がゆっくり上昇し、行1の **+2ab** に衝突する
- 衝突時にバウンス演出（上下に少し弾む）と同時に **+2ab** と **−2ab** が両方消える

### フェーズ8：三角形が消えて c² だけ残る

- 4つの三角形が左上→右上→右下→左下の順に1枚ずつフェードアウトする
- 各三角形の **½ab** ラベルも一緒に消える
- 内側の c² の正方形（赤枠）だけが残る
- 外側正方形の黄色ハイライトも消えて細い白枠になる

### フェーズ9：(a+b)² が消え、a²+b² = c² が完成する

- 行1の **(a+b)²** がフェードアウトする
- 行1に残った **a²** と **b²** が中央に向かって移動し、**a² + b²** としてまとまる
- 内側の c² の正方形の中央に **c²** というラベルがフェードインで表示される
- **c²** ラベルが正方形の中央から数式エリアの **= c²** の位置へスライド移動する
- **= 符号** がフェードインで出現し、**a² + b² = c²** が完成する

### フェーズ10：最終強調

- 完成した式 **a² + b² = c²** が拡大（1.0→1.6→1.3倍）してから落ち着く
- 外側・内側の正方形がフェードアウトし、最後に直角三角形1つと式だけが残る

## デザイン仕様

### カラーパレット（3Blue1Brown風のダークテーマ）
- 背景色：`#0f0f1e`（濃い紺）
- **a**（辺・a²）：`#58c4dd`（シアン青）
- **b**（辺・b²）：`#83c167`（緑）
- **c**（斜辺・c²）：`#e07a5f`（赤橙）
- **(a+b)²**：`#f4d03f`（黄）
- **2ab / ½ab**：`#ffffff`（白）
- 記号・等号：`#ffffff`（白）
- グリッド：薄い白（不透明度4%程度）

### アニメーション全般
- イージング：ease-in-out（滑らかな加減速）
- 全体の長さ目安：60秒前後
- フォント：LaTeX数式フォント（Manimのデフォルトのものを使用）

### 補助要素
- 薄いグリッド線を背景に表示する（透明度低め）
- 直角マーク（□）は各直角三角形の直角頂点に表示する

---

## 実装上の注意

- Manim Community版（`manim` コマンド）を使用すること
- シーン名：`PythagoreanProof`
- 三角形の辺の比は a:b = 105:72 を使用すること（見栄えがよい）
- `MathTex` を使って数式を表示すること
- 式のアニメーションは `Write`、`FadeIn`、`Transform`、`MoveAlongPath` などを活用すること
