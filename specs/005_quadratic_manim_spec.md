## 画面レイアウト

画面を縦に大きく２分割する。

```
┌─────────────────────────┐  ← 1080px 幅
│                         │
│   グラフ領域             │  上端 5% 〜 下端 70%（画面高の約 65%）
│   座標系・放物線         │
│                         │
├─────────────────────────┤  ← 区切り線（薄いグレー）
│   情報パネル             │  上端 71.5% 〜 下端 100%（画面高の約 28%）
│   ① 数式                │
│   ② a / b / c ゲージ    │
└─────────────────────────┘
```

### グラフ領域

- 座標系の中心: 画面横中央・グラフ領域の縦中央
- 表示範囲: x, y ともに −5 〜 5
- グリッド: 1 刻みの細線（`#0f1c2e`）、偶数目盛りにのみ数値を表示
- 座標軸: やや太め（`#263548`）、矢印付き
- 軸ラベル: `x`（右端）・`y`（上端）

### 情報パネル

**① 数式（パネル上部）**

- 中央揃えで表示
- 各項を色分け
  - `y = ` → `#6a8090`
  - `ax²` 部分 → `#38bdf8`（a が未登場のときは `x²` と表示）
  - `- bx` 部分 → `#f472b6`（b 登場後にフェードイン）
  - `+ c` 部分 → `#a78bfa`（c 登場後にフェードイン）
- 係数は小数第2位まで表示（例: `2.00x²`）
- 係数が ±1.00 のときは係数を省略（例: `x²`、`−x²`）

**② パラメータゲージ（パネル中段）**

a・b・c の3本を縦に並べる。

各ゲージの構成（左から順）:
```
[ラベル]  [────────●────────]  [数値]
  a            トラック           1.00
```

- ラベル: 各パラメータの色で表示
- トラック: 濃紺の丸角矩形（`#0e1d2e`）
- フィル: ゼロ位置を起点に現在値まで色付き（不透明度 55%）
- ノブ（●）: 現在値位置に円形、グロー付き
- 数値: ノブ右に小数第2位まで表示
- 未登場のパラメータ（b, c）は全体を薄く表示（不透明度 22%）し、数値は `—` とする

ゲージの値域:
| パラメータ | 最小値 | 最大値 |
|-----------|--------|--------|
| a | −3 | 3 |
| b | −4 | 4 |
| c | −3 | 2 |

## グラフ要素の描画仕様

### 放物線

- 関数: `y = ax² - bx + c`
- 描画色: `#38bdf8`、線幅 3px 相当、グロー（shadowBlur 相当）あり
- グラフ領域内のみクリッピング

### 頂点ドット

- 頂点座標: `(b/(2a),  c - b²/(4a))`
- 白丸（半径 4〜5px 相当）、青いグロー付き
- 常時表示（a が登場した直後から）

### 対称軸

- `x = b/(2a)` の垂直破線（ダッシュ 5px・ギャップ 5px）
- 色: `#f472b6`、不透明度 50%
- ラベル `x=○.○` をグラフ領域上端近くに表示（軸の右側、はみ出しクランプ）
- **b が登場するタイミングでフェードイン**、b 登場前は非表示

## アニメーション仕様

### イージング

全区間に **ease-in-out**（三次または二次のベジェ）を適用する。  
Manim では `rate_func=there_and_back` ではなく `rate_func=smooth` を使用。

### シーン構成とタイムライン

フレーム数は 60fps 基準。`sec = frames / 60` で秒数に換算する。

## Manim 実装ガイドライン

### 推奨クラス構成

```python
class QuadraticBasics(Scene):
    """全シーンを一本のシーンとして実装する"""

    def construct(self):
        self.setup_layout()
        self.scene1_a()
        self.scene2_b()
        self.scene3_c()
        self.scene4_combined()
        self.scene5_summary()
```

### 座標系

```python
# Axes の設定例
axes = Axes(
    x_range=[-5, 5, 1],
    y_range=[-5, 5, 1],
    x_length=<グラフ領域の幅>,
    y_length=<グラフ領域の高さ>,
    axis_config={"color": "#263548", "include_tip": True},
    x_axis_config={"numbers_to_include": [-4,-2,2,4]},
    y_axis_config={"numbers_to_include": [-4,-2,2,4]},
)
```

### パラメータ追跡

`ValueTracker` を a・b・c それぞれに用意し、`always_redraw` で放物線・数式・ゲージを常時更新する。

```python
a_val = ValueTracker(1)
b_val = ValueTracker(0)
c_val = ValueTracker(0)

graph = always_redraw(lambda: axes.plot(
    lambda x: a_val.get_value()*x**2
              - b_val.get_value()*x
              + c_val.get_value(),
    color="#38bdf8",
    stroke_width=3,
))
```

### アニメーション呼び出し例

```python
# a を 1 → −2 へ（Scene 1 の区間 1-4）
self.play(
    a_val.animate.set_value(-2),
    run_time=80/60,
    rate_func=smooth,
)
```

### 情報パネルの数式

`MathTex` を使い、各項を別オブジェクトとして並べて色を適用する。  
係数は `DecimalNumber` を使って ValueTracker と連動させるか、  
`always_redraw` で `MathTex` ごと再生成する。

### ゲージ

`Rectangle`（トラック）＋ `Rectangle`（フィル）＋ `Dot`（ノブ）＋ `DecimalNumber`（数値）で構成し、  
`always_redraw` でノブ位置・フィル幅を ValueTracker に追従させる。

### グロー効果

Manim には shadowBlur 相当の機能がないため、  
同色・低不透明度の太い線を放物線の背面に重ねて擬似的に再現する。

```python
graph_glow = always_redraw(lambda: axes.plot(
    lambda x: ...,
    color="#38bdf8",
    stroke_width=10,
    stroke_opacity=0.2,
))
```

### レンダリングコマンド

```bash
manim -pqh --fps 60 quadratic.py QuadraticBasics
# 縦型出力（1080x1920）の場合は config で解像度を指定
manim -pqh --fps 60 -r 1080,1920 quadratic.py QuadraticBasics
```

または `manim.cfg` に記述:

```ini
[CLI]
pixel_width = 1080
pixel_height = 1920
frame_rate = 60
```
